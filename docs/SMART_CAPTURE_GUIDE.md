# Paste Job Text Guide

Paste Job Text turns copied posting text into an editable draft. It is a deterministic convenience feature, not scraping or AI extraction.

## Use it

1. Copy the useful job posting text from a page, recruiter message, or document.
2. Open **Add Job** and choose **Paste Job Text**.
3. Prepare the review, check the suggested fields and Job Posting Snapshot, and correct anything needed.
4. Set Source and Job Link yourself, then explicitly save.

Paste Job Text works well when a structured import or Browser Capture is unavailable. Browser Capture has documented support for Greenhouse, Indeed, LinkedIn, ZipRecruiter, and Handshake; it still opens the same review-first workflow.

## Keep the right context

The Job Posting Snapshot preserves the employer content. Personal Notes are your own context and remain separate. The parser does not scan pasted text for a link or silently choose a source, so confirm both before saving.

## Job Intelligence Brief

After saving, Application Detail offers **AI Brief**. It analyzes the current approved snapshot only after you explicitly generate it, does not modify captured application fields, and is separate from Paste Job Text extraction. Generated briefs are session-only and require a snapshot of at least 200 characters.
